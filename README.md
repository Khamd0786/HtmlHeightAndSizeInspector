# HTML Height + Size Inspector

A browser-based developer tool for inspecting the rendered height and size of HTML snippets across real mobile device dimensions.

## Features

- 📱 **Device Simulator** — Preview HTML inside accurate phone frames (Pixel, iPhone, Samsung, OnePlus)
- 📐 **Height Measurement** — Measures true rendered content height in dp and raw px
- 📊 **Size Analysis** — Raw size, estimated gzip, DOM depth, element count, word count
- 🔍 **Content Breakdown** — Visual breakdown of HTML tags, inline CSS, JS, and text
- 🤖 **Android Log Output** — Simulated `OnHeightChange` log line for Android WebView debugging
- ✅ **Health Checks** — DOCTYPE, charset, viewport, alt tags, deprecated elements, and more
- 🎯 **Container Padding** — Set left/right padding and see effective width instantly

## Usage

Just open `index.html` in any modern browser — no build step, no dependencies.

1. Select a device from the dropdown
2. Paste your HTML into the editor
3. Click **Render & Analyze**
4. View height, size, and structure metrics instantly
